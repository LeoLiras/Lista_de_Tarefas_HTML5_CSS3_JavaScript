Tarefas

Você precisa escrever uma tarefa primeiro
  • Para acrescentar uma tarefa, basta escrevê-la no campo e clicar no botão ou na tecla enter.
  • Para deletar uma tarefa, basta clicar sobre ela.
  • A cada vez que você envia uma tarefa, todas as tarefas são salvas no storage do navegador. Dessa maneira, mesmo que o navegador seja fechado ou a página seja atualizada, as tarefas ficarão salvas
  • Caso tenha excluído alguma tarefa por engano, basta atualizar a página antes de adicionar outra tarefa
